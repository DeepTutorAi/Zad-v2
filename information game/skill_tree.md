# Skill Tree & Progression System (The Path of Choice)

## Core Concept
*   **Selection System:** แต่ละ Class มีสกิลทั้งหมด **36 สกิล** แบ่งเป็น **12 Tiers**
*   **The Choice:** ในแต่ละ Tier จะมีตัวเลือก **3 สกิล (3 Paths)** ผู้เล่นสามารถเลือกได้เพียง **1 สกิล** เท่านั้น
*   **Max Skills:** ตัวละครหนึ่งตัวจะมีสกิลสูงสุด **12 สกิล** (จาก 36 ตัวเลือก)
*   **Requirements:** การปลดล็อคต้องใช้ **Level** และ **Stat** ที่กำหนด
*   **Strategy:** ผู้เล่นสามารถเลือกผสมสาย (Hybrid) หรือเน้นสายเดียว (Pure) ก็ได้ แต่ต้องวางแผน Stat ให้ถึงเกณฑ์
