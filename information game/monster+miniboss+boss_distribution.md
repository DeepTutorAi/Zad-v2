# Monster Distribution & Raid System

เอกสารนี้ระบุรายชื่อมอนสเตอร์ทั่วไป (General Monsters) ทั้ง 250 ตัว พร้อมระดับความเก่ง (Tier) และระบบการบุกโจมตี (Raid Events) ตามช่วงเวลา

---

## 1. Night Raid & Wave System (ระบบการบุกตามช่วงเวลา)

ระบบนี้จะเริ่มทำงานตั้งแต่ **คืนวันที่ 10** เป็นต้นไป โดยจะมีกลุ่มมอนสเตอร์พิเศษบุกเข้ามาโจมตีผู้เล่น (Ambush) ไม่ว่าจะอยู่ที่ไหน (ยกเว้นใน Safe Zone ที่ปลอดภัยจริงๆ)

### Raid Schedule & Mechanics

| Day Range | Event Type | Enemy Type | Wave Count | Tier Estimate | Description |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Day 10-14** | **First Raids** | General Monsters | 1 Wave (3-5 ตัว) | Tier 3-4 | การรวมตัวของมอนสเตอร์เจ้าถิ่น เริ่มดุร้ายขึ้น |
| **Day 15** | **Double Trouble** | General Monsters | **2 Waves** (ต่อเนื่อง) | Tier 5-6 | Wave 1: 3 ตัว / Wave 2: 4 ตัว (มาซ้อนกันถ้าเคลียร์ช้า) |
| **Day 16-19** | **Escalation** | General Monsters | 1 Wave (4-6 ตัว) | Tier 6-7 | มอนสเตอร์เริ่มมี Elite ปนมาด้วย |
| **Day 20** | **Void Incursion** | **Void Minions** | 1 Wave (3-5 ตัว) | Tier 8-9 | ลูกสมุนของ Void เริ่มปรากฏตัว (Void Walker, Null Shade) |
| **Day 21-24** | **Void Hunting** | Void Minions | 1 Wave (5-6 ตัว) | Tier 9-10 | การไล่ล่าเข้มข้นขึ้น |
| **Day 25** | **Void Onslaught** | **Void Minions** | **2 Waves** (ต่อเนื่อง) | Tier 10-11 | Wave 1: Void Minions / Wave 2: Void Elite (โหดมาก) |
| **Day 26-29** | **Total Despair** | Mixed (Void + Bosses) | Random (1-2 Waves) | Tier 12+ | ช่วงวิกฤต NPC สิ้นหวัง มอนสเตอร์บุกหนักทุกคืน |
| **Day 30** | **The End** | - | - | - | เข้าสู่ Event บอสใหญ่ |
